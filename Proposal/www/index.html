Existing demos and services

iPhylo

The goal of iPhylo is to treat biodiversity objects as equal citizens. Each object has a unique identifier, associated metadata, and is linked to other objects (for example, a specimen is linked to sequences, sequences are linked to publications, etc.). By following the links it is possible to generate new views on existing information, such as a map for study that doesn't have any maps.

I have a demo available. As an example, below is a map of Wang et al. "Phylogeography of the Pygmy Rain Frog (Pristimantis ridens) across the lowland wet forests of isthmian Central America" (doi:10.1016/j.ympev.2008.02.021), generated by extracting the Genbank sequences linked to the paper, and then extracting the museum specimens linked to those sequences. If you don't see the map, use a modern web browser such as Safari 3 or Firefox 2. If you must use Internet Explorer, grab the RENESIS player.

bioGUID

Developed by Joseph Hughes in my lab, Tree Ripper is a first attempt to extract phylogenetic trees from bitmap images.
